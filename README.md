# Market Hisab Railway Telegram Bot

Files:
- bot.py
- requirements.txt
- Procfile

Railway variables:
- BOT_TOKEN
- SPREADSHEET_ID
- GOOGLE_SERVICE_ACCOUNT_JSON
- SCAN_INTERVAL_SECONDS optional, default 60
- SEND_LOW_PERSONAL optional, default true
- SEND_ADMIN_DETAILS optional, default true

Start command:
python bot.py
